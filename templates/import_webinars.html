Импорт вебинаров из Excel

Требования к Excel-файлу:

Файл должен содержать следующие колонки:

  • название вебинара - название вебинара (обязательно)
  • ссылка на вебинар - URL вебинара (обязательно)
  • дата вебинара - дата в формате ДД.ММ.ГГГГ (опционально)
  • номера заданий - список номеров заданий через запятую (например: "1, 2, 5") (опционально)
  • решение прогой или руками - тип решения ("прогой", "руками" или "прогой, руками") (опционально)
  • номер дз - номер домашнего задания (опционально)
  • категория - категория вебинара (1 - обязательный, 2 - повторение, 3 - необязательный) (опционально)
  • категория курса - уровень курса ("Python с нуля", "основной курс", "хард прога", "задание 27") (опционально)
  • комментарий - дополнительная информация (опционально)
  • ссылка на обложку - URL изображения обложки вебинара (опционально)
